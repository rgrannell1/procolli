
# Procolli

Procolli displays files in the proc filesystem as machine-readable JSON.

## Supported Files

```
/proc/buddyinfo
/proc/cmdline
/proc/cpuinfo
/proc/loadavg
/proc/mdstat
/proc/meminfo
/proc/net/dev
/proc/net/protocols
/proc/net/sockstat
/proc/net/sockstat6
/proc/net/stat
/proc/net/tcp
/proc/net/tcp6
/proc/net/udp
/proc/net/udp6
/proc/net/unix
/proc/pressure
/proc/pressure/<resource>
/proc/schedstat
/proc/slabinfo
/proc/stat
/proc/swaps
/proc/vmstat
/proc/zoneinfo
```
